Commands

/adjusttime
Adjust the voice channel time for a member.
Bot permissions: manageMembers
/cleartime
Clear all voice channel times and reset the leaderboard.
Bot permissions: manageChannels
/servergoal
Set your server's activity goals using an interactive modal.
Bot permissions: manageGuild
/noti
Set the channel for voice channel time notifications.
Bot permissions: administrator
/setleaderboardsize
Set the size of the leaderboard.
Bot permissions: manageGuild
/setup
Set up channels for time tracking.
Bot permissions: manageGuild
/setgoalchannel
Set the channel for goal announcements.
Bot permissions: manageChannels
